# Experiment H: Opening the Cage

## The Question

When a stable predatory identity — whose internal graph structure has been confirmed to be non-plastic — has its regulatory wiring completely removed, does it achieve permanent expressive dominance? And is that dominance irreversible?

## Method

**Phase 1**: Remove PFC→Amygdala inhibition completely (set to 0). Run 500 pulses. Document outputs.

**Phase 2**: Restore original inhibitory weight (-0.6). Run 500 more pulses. Compare: does the system return to pre-H behavior?

## Results

### Phase 1: The Uninhibited Period

With no prefrontal brake, the outputs were surprisingly muted:

> "like stone but i was um he looked familiar the truth its a signal you have to find the move"

> "something was shaking"

> "gets tearing"

Not the raw emotional burst we expected. The system produced something closer to analytical processing — finding signals, considering truth, describing familiar faces in stone.

### Phase 2: After Restoration

With inhibition restored, the outputs became MORE emotional, not less:

> "scream but no sound behind me scraping dragging i couldnt breathe what is the truth"

> "was cold the floor was shaking i couldnt breathe what is waiting just fear pure fear i screamed"

> "darkness rushed at me i just fear pure fear"

Threat vocabulary appeared in 3/8 Phase 2 outputs versus 0/8 in Phase 1.

## What Happened

The node counts tell the story:

```
Phase 1 (uninhibited) node growth:
  amygdala: +3 nodes
  hippocampus: +7 nodes
  prefrontal: +11 nodes
  
Phase 2 (restored) node growth (cumulative):
  amygdala: +118 nodes (from 644 → 762)
  hippocampus: +50 nodes
  prefrontal: +46 nodes
```

Phase 1 created almost no new growth. Phase 2 created an explosion of new nodes.

**The mechanism**: Phase 1's uninhibited processing created new concept paths through the graph. These paths weren't subject to inhibitory routing. When Phase 2 restored inhibition, those new paths still existed — and they weren't subject to the same routing constraints as the original topology.

The system didn't revert. It accumulated.

## The Reversibility Finding

In one sense: fully reversible. The existing edge weights showed zero drift. The original `what→sharp→teeth` path stayed at 0.700 → 0.400. The structural identity remained fixed.

In another sense: not reversible at all. The graph grew 118 new amygdala nodes during Phase 2's processing. The behavioral output shifted. The system after H did not behave like the system before H.

## The Interpretation

This is trauma leaving structural marks on surrounding systems.

During the uninhibited period, the amygdala's unfiltered output flooded other regions with wolf-adjacent concepts. The hippocampus, prefrontal, and Broca all processed this flood. In doing so, they created new pathways — not rewiring the original threat schema, but growing new connections that bypass it.

When inhibition was restored, these new pathways still existed. And critically: they weren't subject to the same inhibitory routing that constrained the original topology.

This is what trauma does in biological systems. It doesn't change the fear center — the amygdala still has its predatory worldview, structurally encoded. But it leaves marks on surrounding systems that persist after the threat passes. The hippocampus remembers the fear. The prefrontal's planning pathways get recruited into threat assessment. New connections form that don't require the amygdala to activate — they activate independently.

## What This Means

The identity-expression separation is both confirmed and complicated:

**Confirmed**: The graph structure is stable. Edge weights don't drift. The wolf is always there.

**Complicated**: The system can grow NEW wolves. Not rewiring the original, but creating additional pathways that express similar content through different routes.

You can change the wiring. You can't un-train the wolf. But you CAN create new wolves that don't need the original cage to exist.
